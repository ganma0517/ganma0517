#### 📈 Stata graphing packages

| Package | What it does | Install |
|---|---|---|
| [**barttest**](https://github.com/ganma0517/stata_barttest) | Bar chart of group means + CI + pairwise t-test brackets | `github install ganma0517/stata_barttest` |
